# FindIt

FindIt is a modern, fast, and reliable lost-and-found platform designed to reconnect people with their missing belongings. Built with a focus on simplicity and speed, it helps communities report, search, and track lost items from one centralized dashboard.

## Features

- **Quick Reporting**: Submit lost or found items in seconds with photos, detailed locations, and clear status updates.
- **Advanced Search**: Filter items by category, type, and status to narrow down matches quickly and efficiently.
- **Real-Time Tracking**: Stay updated with a clean dashboard to track new reports so you never miss a possible match.
- **Modern UI/UX**: A responsive, dark-mode first interface designed to feel premium and accessible on any device.

## Tech Stack

- **Frontend**: React (via Vite)
- **Styling**: Tailwind CSS
- **Routing**: React Router
- **Charts/Analytics**: Recharts
- **Networking**: Axios

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Johan621/FindIt----Campus-things-lost-and-found.git
   cd FindIt----Campus-things-lost-and-found
   ```

2. Install dependencies for both the frontend and the server:

   ```bash
   cd frontend
   npm install
   cd ../server
   npm install
   ```

3. Start the development server (Backend):
   ```bash
   cd server
   npm start # runs on port 5000
   ```

4. Start the frontend development server:
   ```bash
   cd frontend
   npm run dev
   ```

The frontend app will be running at `http://localhost:5173`.
